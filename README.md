# 🎤 Speech-to-Text Web Application

A simple and efficient **Speech-to-Text Web App** built using **Flask** and **Python**, which allows users to upload audio files and convert spoken content into text using Google's Speech Recognition API.

---

## 📌 Overview

This project provides a user-friendly web interface where users can upload audio files, and the system processes the audio to generate accurate text output. It leverages the `SpeechRecognition` library for handling speech-to-text conversion.

---

## 🚀 Features

- 🎧 Upload audio files easily through the web interface  
- 📝 Convert speech into text in real-time  
- ⚡ Fast and lightweight Flask backend  
- ❗ Error handling for unclear audio or API issues  
- 🌐 Uses Google Speech Recognition API  

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Speech Processing:** SpeechRecognition  
- **Frontend:** HTML  
- **Other Modules:** tempfile, os
