# Text-to-speech-Transcript-
A Flask-based web application that converts speech audio into text using Google Speech Recognition API.
This project is a Speech-to-Text Web Application built using Python and Flask. It allows users to upload audio files through a simple web interface and converts the spoken content into text using the SpeechRecognition library and Google’s speech recognition service.
The application handles audio processing efficiently by temporarily storing uploaded files and transcribing them in real time. It also includes error handling for unclear audio and API issues, ensuring a smooth user experience.
🚀 Features
Upload audio files via web interface
Convert speech to text instantly
Uses Google Speech Recognition API
Error handling for better accuracy
Lightweight and easy-to-run Flask backend
